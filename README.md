# Bitcoin Tip Sender — OP_NET Styled (Vibecoding Challenge Entry)

A minimal, clean, fully client‑side Bitcoin tipping tool styled in OP_NET colors.

## Features
- ✔ BTC address validation  
- ✔ Static QR generator  
- ✔ Progress bar animation  
- ✔ Copy-to-clipboard  
- ✔ BIP21 Send BTC link  
- ✔ Pure HTML + JS (no backend)  
- ✔ Works on GitHub Pages & VPS  

## How It Works
1. Enter a Bitcoin address  
2. Validate it  
3. Generate QR for mobile wallets  
4. Send BTC via `bitcoin:` BIP21 link
